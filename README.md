## How do I install these formulae?

`brew install k0sproject/tap/<formula>`

Or `brew tap k0sproject/tap` and then `brew install <formula>`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
